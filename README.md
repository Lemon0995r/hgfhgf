Random Quote Generator

**Автор:** [Шигапов Артём Раилевич]

Описание
Простое GUI-приложение на Python, которое выдает случайные вдохновляющие цитаты. 
Программа умеет сохранять историю просмотров в JSON-файл и фильтровать её.

Функционал
- Генерация случайной цитаты из встроенного списка.
- Автоматическое сохранение истории.
- Фильтрация истории по имени автора или теме (категории).
- Проверка корректности данных.

Использование
1. Запустите `main.py`.
2. Нажмите кнопку **"Сгенерировать цитату"**.
3. Чтобы найти цитаты конкретного автора в истории, введите его имя в поле фильтра и нажмите **"Применить"**.
